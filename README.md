# Page Turner

Next.js + Typescript로 만든 도서 검색 프로젝트입니다.
외부 도서 API를 연동해 책을 검색하고 결과를 확인할 수 있습니다.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
